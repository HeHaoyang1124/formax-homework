## A* 搜索算法

- 代码来源：博客园 https://www.cnblogs.com/helloylh/p/17209739.html
- 算法说明：本页代码使用的是A*启发式算法
- 注释说明：本页代码保留了作者原有的格式化风格， 原作者完全使用单行注释，我使用多行注释区分开来与原作者的注释
- 算法思路：
    1. 依次试探八个方向是否能走计算g代价
    2. 保留能走的点，存储在buff数组中
    3. 在buff数组中找到最小代价点（使用优先队列或者维护小根堆的效率更高，但是这作者纯小白），作为下一步拓展点，同时出队列（删除数组元素）
    4. 剩下还是bfs的那一套，直到判断终点（或者已经全部出队列表示死路一条）

## 加州房价回归分析

- 实现过程：大部分通过AI抽卡实现，还需要更多的时间学习和调试
- 文件结构：`linear_regression_manual.py`（模型实现）、`run_plain_regression.py`（普通回归）、`run_ridge_regression.py`（岭回归）
- 算法说明：手动实现多元线性回归，使用正规方程求解；引入L2正则化惩罚项实现岭回归
- 核心思路：
    1. Z-score标准化特征数据，避免量纲影响
    2. 添加偏置列，构建正规方程 (X^T X + λI)w = X^T y
    3. 使用np.linalg.solve求解权重（比直接求逆更稳定）
    4. L2正则化通过λ控制权重大小，防止过拟合（偏置项不参与正则化）
